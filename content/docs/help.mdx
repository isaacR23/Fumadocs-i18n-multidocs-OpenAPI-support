## Real-Time Support

As a paid customer, you have access to:

- 24/7 real-time chat support
- Response time typically under 24 hours
- Email support at [hello@landinganalytics.com](mailto:hello@landinganalytics.com)

## Billing Support

For billing inquiries:

- Use the real-time chat for fastest response
- Email us at [hello@landinganalytics.com](mailto:hello@landinganalytics.com)

To access billing receipts:

1. Go to the application
2. Click your profile in the top right corner
3. Select "Billing"
4. Navigate to "Billing Profile"

## Bug Reports

To report a bug:

1. Visit [our bug tracking system](https://landinganalytics.featurebase.app)
2. Use the provided template
3. Include your project ID (found on the project's main page above the submenu)

### Bug Report Template

```
Project ID: [Your Project ID]
Description: [Detailed description of the issue]
Steps to Reproduce:
1. [First step]
2. [Second step]
3. [Third step]
Expected Behavior: [What should happen]
Actual Behavior: [What actually happens]
Additional Context: [Screenshots, error messages, etc.]
```
