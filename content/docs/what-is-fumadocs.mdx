## Architecture

Fumadocs is built with a modular architecture:

```
fumadocs-core/     # Core functionality and APIs
fumadocs-ui/       # React components and layouts
fumadocs-mdx/      # MDX processing and components
```

## Key Features

- **File-based Routing**: Automatic route generation from your content structure
- **TypeScript Support**: Full TypeScript support with excellent type safety
- **Plugin System**: Extensible architecture with plugins for additional functionality
- **SEO Optimized**: Built-in SEO features and metadata management
- **Accessibility**: WCAG compliant components and keyboard navigation
- **Internationalization**: Multi-language support for global audiences

## Framework Support

Fumadocs works with multiple React frameworks:

- **Next.js**: Full support with App Router and static export
- **React Router**: Compatible with React Router v6
- **Tanstack Start**: Built-in support for Tanstack Start
- **Vite**: Works with Vite-based React applications
